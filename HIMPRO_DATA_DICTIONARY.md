# Himpro HIS Data Dictionary (For KPI Dashboard)

เอกสารนี้รวบรวม **Data Dictionary** ที่เชื่อมโยงระหว่างตารางข้อมูลในระบบ Himpro HIS กับระบบ Dashboard KPI (PDHKPI) เพื่อเป็นคู่มืออ้างอิงในการเขียน Query หรือทำความเข้าใจความสัมพันธ์ของข้อมูลครับ

---

## 1. ข้อมูลผู้ใช้งานระบบ (User & Authentication)

| Database / Schema | Table Name | Column Name | Data Type / Description | การใช้งานใน KPI Dashboard |
| ----------------- | ---------- | ----------- | ----------------------- | ------------------------ |
| `hosdata` | `user` | `userid` | ไอดีผู้ใช้งาน (Primary Key) | ใช้เป็น `$_SESSION['kpi_user_id']` |
| `hosdata` | `user` | `userlogin` | ชื่อผู้ใช้ที่ใช้เข้าสู่ระบบ (Login Name) | ใช้ตรวจสอบตอนเข้าสู่ระบบ |
| `hosdata` | `user` | `username` | ชื่อ-นามสกุลผู้ใช้งาน | แสดงผลชื่อผู้ใช้ในหน้า Dashboard (แปลงจาก TIS-620) |
| `hosdata` | `user` | `password` | รหัสผ่านผู้ใช้งาน | ใช้ตรวจสอบการเข้าสู่ระบบร่วมกับ MD5 |
| `hosdata` | `user` | `npass` / `netpass` | รหัสผ่านแบบเข้ารหัส MD5 | ตรวจสอบรหัสผ่านผู้ใช้ที่มาจาก Himpro |
| `hosdata` | `user` | `groupcode` | รหัสกลุ่มสิทธิ์ผู้ใช้งาน (Role) | ใช้กำหนดสิทธิ์เข้าถึงหน้าต่างๆ |

## 2. ข้อมูลการรับบริการผู้ป่วยนอก (OPD)

| Database / Schema | Table Name | Column Name | Data Type / Description | การใช้งานใน KPI Dashboard |
| ----------------- | ---------- | ----------- | ----------------------- | ------------------------ |
| `opd` | `opd` | `hn` | รหัสประจำตัวผู้ป่วย (Hospital Number) | นับจำนวนผู้รับบริการแยกรายบุคคล |
| `opd` | `opd` | `regdate` | วันที่มารับบริการ | กรองข้อมูลตามปีงบประมาณและรายเดือน |
| `opd` | `opd` | `frequency` | ลำดับการมารับบริการ (Visit No) | นับจำนวนครั้งที่มารับบริการ |
| `opd` | `drug_order_opd` | `codedrug` | รหัสยาที่สั่งจ่าย | เชื่อมกับตาราง `itemlist` เพื่อแยกกลุ่มยา |
| `opd` | `drug_order_opd` | `amount` | จำนวนยาที่สั่งจ่าย | คำนวณยอดการใช้ยา |
| `opd` | `drug_order_opd` | `regdate` | วันที่สั่งจ่ายยา | กรองยอดสั่งจ่ายยาตามวันที่ |
| `opd` | `odiag` | `hn` | รหัสผู้ป่วย | ระบุผู้ป่วยที่ได้รับการวินิจฉัยโรค (NCDs, Communicable) |
| `opd` | `odiag` | `regdate` | วันที่วินิจฉัย | กรองวันที่พบโรค |
| `opd` | `odiag` | `diag` | รหัสโรค (ICD-10) | ค้นหาโรคตามรหัส เช่น E10-E14 (เบาหวาน), I10-I15 (ความดัน) |

| `pcu` | `anc` | `regdate` | วันที่ฝากครรภ์ | เช็ควันที่รับบริการฝากครรภ์ครั้งแรก |

---

> **หมายเหตุการใช้งานฐานข้อมูล:** 
> - **Encoding:** ฐานข้อมูล Himpro HIS (`hosdata`, `opd`, `ipd`, `hos`, `pcu`) เข้ารหัสเป็นแบบ `TIS-620` 
> - การนำข้อมูลที่มีภาษาไทยออกมาใช้งาน (เช่น `username`) ต้องทำการ Convert TIS-620 เป็น UTF-8 ทุกครั้งก่อนแสดงผลบน Web Dashboard
> - **การเชื่อมต่อ DB:** ทุก API File ให้ใช้ `$db = $database->getHisConnection();` ยกเว้นตอนดึงข้อมูล Snapshot หรือ Master KPI ให้ใช้ `$database->getAppConnection();`
